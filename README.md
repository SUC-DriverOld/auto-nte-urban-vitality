# Auto Nte Urban Vitality

这是一个用于《异环》全自动刷“都市活力”的脚本。脚本通过 `pyautogui` 模拟鼠标和键盘操作，目标进程为 `HTGame.exe`。  脚本内使用的点击坐标，基准点为**游戏窗口客户区左上角**，**不包含标题栏和窗口边框**。当前内置坐标以 **1920x1080** 客户区为基准，脚本会自动按比例适配 **1600x900**。对于其他相同宽高比的窗口尺寸，通常也会一起按比例适配。

## 使用前准备

运行脚本前，用户需要手动完成以下准备：

1. 进入游戏，并停留在任意一个“一咖舍”前台页面。
2. 将关卡切换到 `2-8 45秒`。
3. 确保页面上的 `F` 按键提示当前指向的是 `店长特供`。
4. 保持游戏页面状态不要变化，再启动脚本。
5. 启动脚本后有10秒倒计时，在此期间请返回游戏窗口，任意点击一次游戏窗口以确保它处于前台并获得焦点。

> [!IMPORTANT]
> 1. **务必使用管理员权限运行**
> 2. 如需终止脚本，直接关闭控制台窗口即可，或者在控制台内按 `Ctrl+C` 强制中断。
> 3. 运行过程中请不要随意移动鼠标及游戏窗口。
> 4. 使用娜娜莉，白藏，海月三人组，并且都升级一下都市技能（最好都升满）
> 5. **作者不对使用本脚本造成的任何后果负责，请谨慎使用！**。

## 运行方式

建议从release页面直接下载打包好的 `exe` 文件，然后以管理员方式运行。也可以从源码运行，或者自己打包为 `exe`。

### 源码运行

使用 `uv` 直接运行：

```powershell
uv sync
uv run auto-nte
```

### 打包为 EXE

如果需要打包为独立的 `exe`，可以按照下面的步骤：

```powershell
uv sync --group dev
.\build.ps1
```

打包完成后，产物默认位于：`dist/auto-nte.exe`
